🛠️ 飞书新群无响应排障指南 (Troubleshooting)

如果机器人在新的飞书群内拉起后没有响应，请按顺序检查：

  • 核对上表的 Peer ID 必须与要绑定的新飞书群 Chat ID 完全一致 (通常以 oc_ 开头)。
  • 修改 openclaw.json 后，请去 Settings 页点击 重启 OpenClaw 核心 使动态路由生效。
  • 前往 System Logs (系统日志) 检查飞书插件是否收到了 Webhook 网络回调。
  • 刚入群的机器人至少需要 被明确 @ 提及一次 才能获取读取上下文和响应的权限。
